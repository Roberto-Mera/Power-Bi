# Proyecto 1

### Objetivo: 
Desarrollar un reporte interactivo que nos permita analizar la ventas de videojuegos (en millones) a nivel mundial.

Se requiere poder analizar por las categorías de:

- Año
- Región
- Plataforma
- Género
- Editorial
